# HouseHunt

HouseHunt is a MERN Stack House Rent Management System.

## Features

- User Registration and Login
- Role-Based Access (Admin, Owner, Renter)
- Add Property
- View Properties
- Property Booking
- Admin Management
- MongoDB Atlas Integration

## Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## Run Project

### Backend

cd server

npm install

node server.js

### Frontend

cd client

npm install

npm run dev
